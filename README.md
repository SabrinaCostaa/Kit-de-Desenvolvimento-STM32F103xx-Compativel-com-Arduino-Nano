# Kit-de-Desenvolvimento-STM32F103xx-Compativel-com-Arduino-Nano
Projeto de hardware para um Kit de Desenvolvimento ARM Cortex-M3 (STM32F103xx). Replica o Arduino Nano em pinagem e dimensões, mas com performance 32-bit (72MHz). Integra gerenciamento de clock duplo (HSE/LSE), USB nativo (VCP/DFU) e três fontes de alimentação (7-15V, 5V, 3.3V).
